接入检查

现阶段功能直达开放了微信公众平台公众号后台接入流程，详情可以查看微信公众平台公众号后台接入流程。在api正式接入之前，开发者可以调用接口来检查小程序之前是否接入过功能直达，若接入，该接口会返回当前小程序接入状态，若未接入，该接口会返回该小程序是否允许接入功能直达。

接口说明

接口调用基本信息 说明
协议 https
http请求方式 POST
请求URL https://api.weixin.qq.com/widget/funcwidget/getfuncwidgetdetail?access_token=ACCESS_TOKEN
POST数据格式 json
接口能力 检查当前小程序接入状态。

请求数据格式

无

基本字段 字段类型 说明
can_apply bool 该小程序是否允许申请功能直达，若否，会填入reason字段原因
reason string 不允许接入的原因
func_list array 该小程序下申请的功能列表
commit_times int 一个自然月内提交次数
left_commit_times int 一个自然月内剩余提交次数
has_bind_gzh bool 该小程序之前是否在公众号mp端申请过
bind_appid string 申请的公众号appid
bind_nickname string 申请的公众号名称
pages array 可以用于作为跳转路径的jumpUrl

func_list下字段说明：

func_list字段 字段类型 说明
keywords string 提交的功能词，用于搜索侧召回
areacode array 功能服务覆盖的地域列表，取值看地域编码文档
brandname string 功能直达服务的品牌名，若填空，则取小程序名称
categorylist array 该功能所属类目列表，每个选择类目至少需要填入一级和二级类目，具体类目ID见类目树接口
material object 该功能对应的素材
online_keywords string 前一次通过审核的功能词
online_areacode array 前一次通过审核的地域列表
online_categorylist array 前一次通过审核的类目
online_brandname string 前一次通过审核的品牌名
online_material object 前一次通过审核的素材
status string 审核状态
reason string 若审核不通过，这里会写明原因
index int 提交审核的功能序号，修改时需要填入

categorylist字段说明：

categorylist字段 字段类型 说明
id1 int 一级类目ID
id2 int 二级类目ID
id3 int 三级类目ID，0表示未选择三级类目

material下字段说明

material字段 字段类型 说明
iconType int 0为不设置icon，1为设置icon
items array 每一个item对应一个宫格数据，可以设置2个或4个宫格

items array下字段说明

items字段 字段类型 说明
icon string icon图片url
title string 宫格的标题
desc string 宫格的描述
jumpUrl string 点击宫格的跳转路径
错误码 说明
9009099 系统失败

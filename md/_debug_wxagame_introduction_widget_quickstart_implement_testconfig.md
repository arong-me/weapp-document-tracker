# 配置后台测试接口

开发者可在**模板与组件 - Widget - 模板管理 - 模板申请单 - 实现模板 - 配置后台测试接口**中配置测试环境后台接口。配置完成后，微信后台将请求发送至配置的测试接口，小程序体验名单内的人员可以方便地在微信客户端体验和测试。此处所配置的测试接口仅用于开发者进行体验和测试，与生产环境的接口并不一定相同。生产环境的接口配置见[接入概述](https://developers.weixin.qq.com/miniprogram/dev/api/custommsg/callback_help.html?t=18092916)。

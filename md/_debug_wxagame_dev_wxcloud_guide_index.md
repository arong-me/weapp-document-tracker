# 开发指引

云开发提供了一整套云服务及简单、易用的 API 和管理界面，以尽可能降低后端开发成本，让开发者能够专注于核心业务逻辑的开发、尽可能轻松的完成后端的操作和管理。

下面我们将分一下部分介绍如何上手使用云能力：

*   [云开发控制台](console.html)：可视化的云资源管理器
*   [初始化](init.html)：云 API 的初始化
*   [数据库开发指引](database.html)：如何上手数据库开发
*   [存储开发指引](storage.html)：如何上手云存储开发
*   [云函数开发指引](functions.html)：如何上手云函数开发

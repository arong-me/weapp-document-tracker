# 插件调用 API 的限制

插件可以调用的 API 与小程序不同，主要有两个区别：

*   插件的请求域名列表与小程序相互独立；
*   一些 API 不允许插件调用（这些函数不存在于 `wx` 对象下）。

`wx.login` 和 `wx.requestPayment` 在插件中不能使用，但可以通过插件功能页来 **获得用户信息** 和 **进行支付** ，请参考[插件功能页](functional-pages.html)。

目前，允许插件调用的 API 及其对应版本要求如下：

# 基础库更新日志

[查看开发者工具更新日志](../devtools/uplog.html)

[查看云开发更新日志](../wxcloud/reference/changelog.html)

### v2.4.0 (2018.11.05)

1.  `A` 新增 框架 支持在手机上启用屏幕旋转 [详情](view/resizable.html#在手机上启用屏幕旋转支持)
2.  `A` 新增 插件 支持收货地址功能页 [详情](plugin/functional-pages/choose-address.html)
3.  `A` 新增 插件 增加界面背景设置接口 `wx.setBackgroundTextStyle`、`wx.setBackgroundColor` [详情](plugin/api-limit.html#背景)
4.  `A` 新增 API mDNS 局域网通信能力 [详情](ability/mDNS.html)
5.  `A` 新增 API 动态消息能力 [详情](open-ability/share/updatable-message.html)
6.  `A` 新增 API 图片压缩接口 `wx.compressImage` [详情](../api/media/image/wx.compressImage.html)
7.  `U` 更新 框架 组件属性设置支持 rpx
8.  `U` 更新 插件 插件功能页可以在所有者小程序中跳转
9.  `U` 更新 API `wx.onCompassChange` 支持获取精度 [详情](../api/device/compass/wx.onCompassChange.html)
10.  `U` 更新 组件 `<camera>` 支持解析二维码 [详情](../component/camera.html)
11.  `U` 更新 组件 `<video>` 获取缓冲数据长度 [详情](../component/video.html)
12.  `U` 更新 组件 `<live-pusher>` 直播组件支持混音 [详情](../api/media/live/LivePusherContext.html)
13.  `U` 更新 组件 `<video>` 快进手势优化
14.  `F` 修复 框架 `Widget`调试日志无法正确输出问题
15.  `F` 修复 框架 iOS 下打开 7 个页面以上，然后一次性 `wx.navigateBack` 到首页时，会导致白屏的问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/000ea0d6858f385cb3b6f853f56000)
16.  `F` 修复 框架 安卓对同一个资源同时下载导致写文件冲突的问题
17.  `F` 修复 框架 iOS 下 `App.onPageNotFound` 会先出现原生 404 页，再推入正确页面的问题
18.  `F` 修复 框架 iOS 下二维码扫码传参读取不一致的问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/000cccc4fd4520246686de6b051c00)
19.  `F` 修复 插件 加载分包的时候插件被重新加载的问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/000a0e07970308ed0b775ba3751c00)
20.  `F` 修复 API `chooseImage` iOS 下会把 png 图片改成 jpg 图片的问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/000020703cce90ad3cd6da9a951800)
21.  `F` 修复 API `BackgroundAudioManager` iOS下 pause 返回了 0 和 1，没有对齐工具和安卓返回 bool 的问题
22.  `U` 更新 API `BackgroundAudioManager` 安卓不支持本地文件的问题
23.  `F` 修复 API `wx.request` iOS 下请求数据 UTF8 解码失败时不直接回调 fail 的问题
24.  `F` 修复 API `wx.createInnerAudioContext` 安卓下小程序跳转小程序之后调用无效的问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/000c06f3d84600b896b6c50ba5fc00)
25.  `F` 修复 组件 `<input>` 进入页面时设置聚焦导致 iOS 下导航栏下落的问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/000ce289f008a0c9e437a792e51c00)
26.  `F` 修复 组件 `<input>` 进入页面时设置聚焦导致安卓固定在底部的输入框被键盘挡住的问题
27.  `F` 修复 组件 `<input>` 在 iOS 12 beta 下双击会隐藏键盘的问题
28.  `F` 修复 组件 `<input>` 安卓下光标放置在最前面时获取内容为空的问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/000e824805c778b810175857a5b000)
29.  `F` 修复 组件 `<input>` 安卓下 color 设置 rgba 导致获取内容为空的问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/000268682848185a081799ed156400)
30.  `F` 修复 组件 `<input>` iOS 下内容为空时删除操作不触发事件的问题
31.  `F` 修复 组件 `<input>` 安卓下 type 为 number/digits 时键盘挡住输入框问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/0002aa4b1dcae87099d6567b451800)
32.  `F` 修复 组件 `<textarea>` 三星输入法英文状态输入时，不触发 `bindinput` 问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/4d03b947663620fb99e74041307f7f8c)
33.  `F` 修复 组件 `<textarea>` iOS 点击输入法的表情符号时键盘消失并失去焦点问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/000a088cf38330f83a470ea065a400)
34.  `F` 修复 组件 `<textarea>` iOS 点击键盘表情切换出现黑屏的问题 [反馈详情](https://developers.weixin.qq.com/community/develop/doc/0008ca6a6cc69092ab47c12195b400)
